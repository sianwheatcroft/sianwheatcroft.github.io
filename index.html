CSS

Tag Description Example
font-family tells browser what fonts to use font-family{
Helvetica, serif, Ariel;
}
font-size tells the browser what size font to use font-size{
18pt;
}
text-decoration applies a decoration or clears it from a piece of text. none, underline, overline, line-through or blink can be used. text-decoration{
none;
}
text-align tells the browser where to align the text. start, end, left, right, center or justify can be used. text-align{
center;
}
color changes the color of the font color{
white;
}
list-style-type tells the browser how a list item will be render (the bullet point) google this as there are many options! list-style-type{
various options;
}
display helps to position elements on the page. google this as there are many options. display{
inline-block;
}
padding applies padding around an element. top, left, bottom, middle can be used. padding{
10px 20px 20px 10px;
}
margin applies a margin to an element. top, left, bottom, middle can be used. margin{
10px 20px 20px 10px;
}
border applies a border to an element. top, left, bottom, middle can be used. border{
solid white 10px;
}
